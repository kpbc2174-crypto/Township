Township v1.0.15

This build adds the first automatic building on prepared Township Lot Marker lots.

Test path:
1. Activate Township BP/RP.
2. Place Township Founding Stone.
3. Wait for the Township Builder and starter camp.
4. Place a Township Lot Marker inside the township build radius.
5. Let the builder prepare the lot and then build the Small House.
6. Crouch near the Lot Marker to view lot status.
