Township v1.0.58 Changelog

Purpose
- Emergency rollback/stabilization build after v1.0.57 broke block behavior.

Changes
- Restored the Behavior Pack and Resource Pack content from the verified v1.0.56 base.
- Bumped BP and RP manifest versions to 1.0.58 for phone-safe import/update.
- Preserved all existing UUIDs.
- Removed the unverified v1.0.57 saved-building/lot-marker build-selection code by reverting to v1.0.56 code.

What this build does NOT include
- Does not include the v1.0.57 lot-marker saved-building feature.
- Does not attempt any new recorder or lot-copy logic.

Test focus
- Verify all custom blocks appear again.
- Verify Founding Stone click UI opens.
- Verify Lot Marker / Build Lot / Ground Leveler click UI opens.
- Verify existing township behavior from v1.0.56 is restored.
